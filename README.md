# Création d'une API avec Express

Prérequis :

- NodeJS
- Npm

Installation :

```bash
git clone git@github.com:nico-develop/api-express.git
cd api-express
npm i
```

Démarrage :

```bash
npm start
```

Accès à l'API sur l'url : `http://locahost:3000`

# Urls de l'API

| VERBES | URI        | 
|--------|------------|
| GET    | /users     |
| GET    | /users/:id |
| POST   | /users     |
| PUT    | /users/:id |
| DELETE | /users/id  |
